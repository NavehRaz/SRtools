# SRtools.extras

This subpackage contains additional classes and functionality for SRtools.

## Purpose

Use this subpackage to add:
- Experimental features
- Extended utilities
- Additional analysis tools
- Custom classes that extend core functionality

## Structure

Add your extra modules here. Each module should be a `.py` file containing 
related classes and functions. Import them in `__init__.py` to make them 
available from the `SRtools.extras` namespace.

## Example Usage

```python
from SRtools.extras import YourExtraClass
```
